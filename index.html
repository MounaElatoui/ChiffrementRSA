🔐 RSA - Cryptographie à Clé Publique

Conforme au cours : Chiffrement C = Me mod n | Déchiffrement M = Cd mod n

⚙️ 1. Génération des clés RSA

Choisissez deux nombres premiers p et q, puis un exposant e

🔒 2. Chiffrement

Entrez votre message en clair (lettres, espaces, ponctuation)

🔓 3. Déchiffrement

Collez le message chiffré reçu

📚 RSA - Rivest, Shamir, Adleman (1978) | Basé sur la difficulté de factorisation de grands nombres

Formules : C = M^e mod n | M = C^d mod n | e × d ≡ 1 mod φ(n) avec φ(n) = (p-1)×(q-1)
